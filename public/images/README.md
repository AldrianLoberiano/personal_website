# Personal Website - Image Assets

Place your project images in this directory:

- `project1.jpg` - Screenshot or image for Project One
- `project2.jpg` - Screenshot or image for Project Two
- `project3.jpg` - Screenshot or image for Project Three
- Add any other images you need for your website

Recommended image sizes:

- Project images: 800x600px or similar aspect ratio
- Profile photo: 400x400px (square)
- General images: Optimize for web (compress for faster loading)
